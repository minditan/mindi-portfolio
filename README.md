# Mindi Portfolio (Three.js)

## Setup

1. Put your resume PDF at `public/resume.pdf`
2. `npm install`
3. `npm run dev`
4. Open http://localhost:5173

## Deploy

`npm run build` → upload `dist/` to Vercel/Netlify/GitHub Pages.

## Click resume

Clicks on the desk/computer area open the PDF. Adjust `deskZone` in `src/main.js` if the hit area feels off.
